# Random Walk

## Descrição

Um modelo gráfico que apresenta uma trajetória composta por uma sucessão de passos aleatórios no espaço.
Marcado com vermelho no ponto inicial e verde no final.

## Tecnologias utilizadas

- Python 3
- Jupyter Notebook

## Instalação

1. Clone o repositório:

```bash
git clone https://github.com/JuliaGabP/RandomWalk.git
```

2. Instale as dependências:

```bash
pip install notebook
```

## Como executar o Random

1. Ative o notebook:

```bash
jupyter notebook

```
2. Localize e abra no navegador o arquivo Untitled.ipynb dentro da pasta do repositório.  

3. Execute as células em ordem.
